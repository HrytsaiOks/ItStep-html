HTML теги и их описание

<h1> - Заголовок 1 уровня.

Используется для обозначения основного заголовка страницы. Самый важный заголовок.

<h2> - Заголовок 2 уровня.

Используется для деления содержания на разделы.

<h3> - Заголовок 3 уровня.

Заголовок внутри <h2> Меньший по значимости.

<h4> - Заголовок 4 уровня.

Заголовок внутри <h3>, используется для ещё более детального деления.

<h5> - Заголовок 5 уровня.

Меньше по значимости. Используется редко, для очень вложенных структур.

<p> - Абзац текста.

Создаёт отступы между блоками текста.

<br> - Перенос строки.

Пример: Это строка с <br> переносом строки.

Вставляет перенос строки внутри абзаца, без создания нового параграфа.

<strong> Делает текст жирным и придаёт ему семантическую важность.

Пример: <strong>важный текст</strong>

<em> - Выделение (курсив)

Пример: <em>акцентированный текст</em>

Делает текст курсивным и выделяет интонацию.

<ol> и <li> - Нумерованный список

Пример: <ol>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
</ol>

Создаёт список с номерами. Каждый пункт задаётся через тег <li>.

<ul> и <li> - Маркированный список

Пример: <ul>
  <li>Элемент</li>
  <li>Элемент</li>
</ul>

Список с точками вместо номеров.

<div> - Контейнер

Пример:<div>Это блок-контейнер</div>

Универсальный блок для группировки элементов.
